# ForgeX 技术栈

## 前端
- **框架**: Vue 3 (Composition API + `<script setup>`)
- **构建**: Vite 5
- **语言**: TypeScript
- **路由**: Vue Router 4
- **状态**: Pinia
- **样式**: CSS Custom Properties + 组件内 scoped CSS
- **图标**: Lucide Icons

## 部署
- **托管**: GitHub Pages
- **CI**: GitHub Actions (自动构建部署)
- **容器**: Docker + Nginx（备选自托管方案）

## 开发规范
- 包管理: pnpm
- 代码风格: ESLint + Prettier
- 提交规范: Conventional Commits
